# Planning-Pack Handoff Specification

| Field | Value |
|---|---|
| Package | `planning-pack` |
| Canonical skill | `architecture` |
| Unit | `architecture/default` |
| Runtime dependency | Codex collaboration is optional; repository and file inspection remain available without it |
| External research | Optional, bounded to three official primary-source page fetches |
| Writes | One contained `3-architecture.md` lifecycle document only |
| Git authority | Read-only |
| Core discovery | Forbidden; this payload remains outside the core plugin manifest and skill directory |

A later separate-plugin repository must provide its own manifest, dependency declaration, installation test, review/verification gates, and release authorization. Pack-ready evidence in this repository proves transferable payload integrity only; it does not claim installation or publication.
