# Planning-Pack Handoff Specification

| Field | Value |
|---|---|
| Package | `planning-pack` |
| Canonical skill | `feasibility-study` |
| Unit | `feasibility-study/default` |
| Runtime dependency | Codex collaboration is optional; repository inspection works locally |
| External research | Optional, bounded to three official primary-source page fetches |
| Writes | Response-only by default; one explicitly named contained Markdown report when requested |
| Git authority | Read-only |
| External systems | Read-only research; no connector setup or mutation |
| Core discovery | Forbidden |

A later separate-plugin repository must provide its own manifest, dependencies, installation tests, review/verification gates, and release authorization. This handoff proves transferable payload integrity only.
