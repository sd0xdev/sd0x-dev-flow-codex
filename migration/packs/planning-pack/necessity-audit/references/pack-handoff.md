# Planning-Pack Handoff Specification

| Field | Value |
|---|---|
| Package | `planning-pack` |
| Canonical skill | `necessity-audit` |
| Unit | `necessity-audit/default` |
| Runtime dependency | One optional read-only challenger; local analysis remains available |
| External research | Optional, bounded to three official primary-source page fetches |
| Writes | None; report is response-only |
| Git authority | Read-only |
| External systems | Read-only research only |
| Core discovery | Forbidden |

A later separate-plugin repository must provide its own manifest, dependencies, installation tests, review/verification gates, and release authorization. This handoff does not claim publication or live installation.
